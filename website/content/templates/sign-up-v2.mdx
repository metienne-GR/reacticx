## Features

- Animated grid background with configurable cell size, line color, and width
- Vignette effect and top fade overlay rendered with Skia
- Large "Create Account" heading (42px) with dark theme styling
- Email and password inputs with dark backgrounds (#121212)
- Eye icon toggle for password visibility
- "Forgot Password?" link with absolute positioning
- Terms and Conditions checkbox
- Social login buttons for Apple and Google
- Full rounded sign-up button (borderRadius: 99)
- Responsive layout with `useResponsive` hook

## Dependencies

| Package | Purpose |
|---------|---------|
| `@shopify/react-native-skia` | Canvas rendering for the animated grid and vignette |
| `expo-font` | Custom font loading |
| `expo-vector-icons` | AntDesign and Ionicons for input icons |
| `react-native-safe-area-context` | Safe area insets handling |
