## Features

- Parallax image carousel header with swipeable property photos and page counter pill
- Animated sticky top nav bar that appears on scroll (back, share, save buttons)
- Floating header buttons with frosted-glass circle style
- Heart/save toggle with pink highlight on active state
- Rating bar with star rating, Guest Favorite laurel wreath badge, and review count
- Host row with avatar, superhost badge, and years hosting
- Highlights section (trophy, key, calendar) with icon and subtitle
- Full description with a "Show more" button
- Amenities list with SF Symbol icons and "Show all" button
- "Where you'll be" section with map placeholder
- Sticky bottom reserve bar with strikethrough original price, total price, free cancellation badge, and gradient Reserve button
- Responsive layout via `useResponsive` hook (`rf`, `rv` scaling)
- iOS SF Symbols for all icons with safe area inset handling

## Dependencies

| Package | Purpose |
|---------|---------|
| `expo-font` | Custom font loading (SF Pro Rounded, Helvetica Now Display) |
| `expo-symbols` | iOS native SF Symbols for all icons |
| `expo-linear-gradient` | Gradient fill on the Reserve button |
| `expo-router` | Navigation (`useRouter`, `Stack.Screen`) |
| `react-native-safe-area-context` | Safe area insets for bottom bar and header |
| `@hugeicons/react-native` | Laurel wreath icons for Guest Favorite badge |
| `@hugeicons/core-free-icons` | Icon definitions (`LaurelWreathLeft01Icon`, `LaurelWreathRight01Icon`) |
| `@expo/vector-icons` | Ionicons checkmark in the Free Cancellation badge |

> The component also depends on the internal `parallax-header` template and the `use-responsive` helper hook, which are included when you install via the CLI.
