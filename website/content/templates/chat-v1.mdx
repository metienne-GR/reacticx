## Features

- AI-style animated text streaming with character-by-character fade-in (18ms delay, 250ms duration)
- Custom dark message bubbles with sender/receiver styling
- Expanding multiline input bar (up to 5 lines)
- Voice button that transforms into a send arrow when text is entered
- Empty state with greeting message and quick-action feature chips (Surprise me, Transform image, Learn maths, Learn coding)
- Secondary action icons (+, globe, bullhorn) alongside the input
- Dynamic layout animations for smooth message transitions
- iOS SF Symbol support with Android fallback
- Dark theme optimized UI

## Dependencies

| Package | Purpose |
|---------|---------|
| `react-native-gifted-chat` | Chat UI framework with message list and composer |
| `react-native-reanimated` | Powers the text streaming fade-in animation |
| `expo-font` | Custom font loading (SF Pro Rounded, Helvetica Now Display) |
| `expo-symbols` | iOS native SF Symbols for icons |
| `expo-vector-icons` | MaterialCommunityIcons, Ionicons, FontAwesome icons |
