## Features

- Purple radial gradient background with 7-color stop points rendered via Skia Canvas
- Email, password, and confirm password input fields with custom validation styling
- Social login buttons for Google and GitHub with custom SVG icons
- "OR" divider with dynamically calculated width
- Responsive text sizing using the `useResponsive` hook
- Custom fonts (SF Pro Rounded, Helvetica Now Display)
- Purple accent colors for focused inputs (#d1a0f2)
- Safe area aware layout
