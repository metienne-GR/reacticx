## Writing a Good Request

<Steps>

<Step>

### Component Name

Give it a clear, descriptive name. Think about what it does or how it behaves.

<div className="not-prose mt-3">
  <p className="font-mono text-[11px] text-zinc-600 bg-zinc-900/40 border border-zinc-800/60 rounded-lg px-3 py-2 inline-block">
    e.g. "Magnetic Haptic Button" or "Liquid Progress Ring"
  </p>
</div>

</Step>

<Step>

### Description

What it does, how it behaves, and what makes it unique. Be specific about the animations and interactions you have in mind.

</Step>

<Step>

### Use Case

Describe the real-world scenario where this component would be useful. This helps us understand the value and prioritize accordingly.

</Step>

<Step>

### Reference

Share anything visual — a link, screenshot, video, Dribbble shot, or CodeSandbox. Even rough inspiration helps.

</Step>

<Step>

### Priority

  <div className="space-y-2">
    <div className="flex items-center gap-3 rounded-xl border border-zinc-800/60 bg-zinc-900/30 px-4 py-3 backdrop-blur-sm">

      <span className="text-sm font-medium text-zinc-300">Nice to have</span>

  </div>
</div>

</Step>
